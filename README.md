# math en jean 

ce repo contient le code fait pour notre projet pour math en jean (https://www.mathenjeans.fr/).
math en jean est un projet scolaire ou des mathématicien sont mis en contact avec des éleve (du collège jusqu'au lycée) pour leur proposer des situations de recherche mathématique.

# notre projet
notre projet est "simple" nous cherchons a trouver un maximum de méthodes pour trouver le centre d'un nuage de point. 

pour l'instant nous avons trouver X méthodes différentes:

- [x] méthode par la moyenne
- [x] méthode par le barycentre 
- [x] méthode par le "point déja présent"
- [x] méthode de l'enveloppe convexe
- [x] méthode par projection
- [x] méthode du centre des gravitées

# installation
créer un environnement virtuel python:

`python -m venv . `

installer les dépendances:

`./bin/python ./bin/pip install -r requirements.txt`

et s'amuser:

`./bin/python main.py `
